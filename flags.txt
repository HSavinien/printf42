list of printf flags :

* 'c'	display a char
* 's'	display a string
* 'p'	display an address
* 'd'	display a signed number
* 'i'	display a signed number
* 'u'	display an unsigned number
* 'x'	display a number in hexa base, with lowercase letters
* 'X'	display a number in hexa base, with upperercase letters
* '%'	display the '%' char

* '-'	
* '0'	
* '.'
